Just like other public clouds (Azure, AWS), Google Cloud also has a huge catalog of products for supporting different use cases. Here's a very brief overview of a few popular products that may be a good starting point for exploration:
## Running Code

 - **Compute Engine** : Create and run virtual machines on Google infrastructure.
 - **Cloud Run** : Deploy containerized applications on a fully managed serverless platform
 - [[Cloud Run Functions]] : Lightweight compute solution to create single-purpose, stand-alone functions that respond to Cloud events without the need to manage a server or runtime environment. 
 - **App Engine** : Fully managed, serverless platform for developing and hosting web applications at scale.

## Storing Data

- **Cloud Storage** : Object/Blob storage solution, good for storing unstructured data like audio, video and images.
- **Cloud SQL** : Managed versions of MySQL, Postgres and SQL Server databases
- **Cloud Firestore** : NoSQL document database (similar to MongoDB)

## AI/ML

- **Cloud Vision AI** : supports features like image labeling, face and landmark detection, OCR.
- **Cloud Natural Language** : NLP solutions like sentiment analysis, entity analysis, entity sentiment analysis, content classification.
- **Vertex AI** : Build, deploy and train ML models.
