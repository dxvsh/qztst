**Problem Statement**

Design a stack that supports push, pop, top, and retrieving the minimum element in constant time.

Implement the `MinStack` class:

- `MinStack()` initializes the stack object.
- `void push(int val)` pushes the element `val` onto the stack.
- `void pop()` removes the element on the top of the stack.
- `int top()` gets the top element of the stack.
- `int getMin()` retrieves the minimum element in the stack.

You must implement a solution with `O(1)` time complexity for each function.

**Example Input**
`["MinStack","push","push","push","getMin","pop","top","getMin"]`
`[[],[-2],[0],[-3],[],[],[],[]]`

**Explanation**
```python
MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.getMin(); // return -3
minStack.pop();
minStack.top();    // return 0
minStack.getMin(); // return -2
```

**Solution**
This is just a standard stack implementation apart from the `getMin` functionality. That is the only tricky part in this problem. We need to implement the stack in a way such that the smallest element in the stack can be retrieved in $O(1)$ time.
To achieve this, we can maintain two stacks. One would be the normal stack to track the elements that are getting pushed and popped. The other one would keep track of the smallest element at that step in time.
At each push, we'll be pushing into both the stacks. The min_stack will store the current smallest element corresponding to each node of the actual stack.

```python
class MinStack:

    def __init__(self):
        self.stack = []
        self.min_stack = []
        
    def push(self, val: int) -> None:
        self.stack.append(val)
        # we find the minimum value at the current time step
        # if this new val is smaller than the current smallest
        # element, we place it on top of the min stack
        min_val = min(val, self.min_stack[-1] if self.min_stack else val)
        self.min_stack.append(min_val)
        
    def pop(self) -> None:
        self.min_stack.pop()
        return self.stack.pop()
        
    def top(self) -> int:
        return self.stack[-1]
        
    def getMin(self) -> int:
        return self.min_stack[-1]
```
